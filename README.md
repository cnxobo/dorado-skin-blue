# dorado-skin-blue
> dorado7 的皮肤组件
## 使用方式
* 安装到maven本地仓库

```
mvn install
```

* pom 引入依赖

```
<dependency>
  <groupId>org.xobo.dorado</groupId>
  <artifactId>dorado-skin-blue</artifactId>
  <version>0.0.1-SNAPSHOT</version>
</dependency>
```

* dorado配置文件（configure.properties）指定皮肤

```
view.skin=xoboblue
```
